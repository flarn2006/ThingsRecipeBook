Things Recipe Book

Start a new Things game

Click here if you haven't already to install the user script. You will need Greasemonkey or (untested) Tampermonkey, so get that first if you don't have it. If the user script is not installed, no recipe or game data will be recorded.

To look up a recipe for a Thing, start typing its name in the search box above. You can also click the name of an ingredient to look up its recipe(s). Provided the user script is working properly, new recipes will automatically be added as you discover them.

To browse previously-played games containing a specific Thing, click its name in the leftmost column. Then, click the game's ID to open it in a new tab.

Have fun!
